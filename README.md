# Notbody_Plugins
## autoapply_siteurl
这是一款定时自动提交网页链接到 百度资源搜索平台 的插件

## comment_ipaddr
这是一款适用于 Argon主题 的插件（WordPress博客系统），开启后可在评论区显示评论者的IP归属地
### 使用方法
1. 下载“comment_ipaddr”至WordPress的插件目录。
2. 在管理员后台插件目录内启用即可。
3. 展示归属地的图标可自行在comment_ipaddr.php内根据提示修改
